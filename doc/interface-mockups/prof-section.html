Intro to Internet Programming

This course provides students with an overview of Internet and World Wide Web programming, development tools, resources, utilities, multimedia, and security issues. Students gain hands-on experience in Hypertext Markup Language (HTML) and a scripting language such as JavaScript. Time permitting, we will also investigate Rich Internet Applications (RIAs), and Web services.
